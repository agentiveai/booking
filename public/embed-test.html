Type 1 Inline Embed Widget

The inline embed displays the booking widget directly embedded in your page content. Perfect for dedicated booking pages or sections.

<!-- Add this where you want the widget to appear -->
<div id="booking-widget" data-provider-id="YOUR_USER_ID"></div>
<script src="https://yourdomain.com/widget.js"></script>
Live Demo

Type 2 Popup Button Widget

Add a custom button anywhere on your page that opens the booking widget in a beautiful modal popup. Great for navigation bars, hero sections, or call-to-action areas.

<!-- Add this button anywhere on your page -->
<button data-booking-popup="YOUR_USER_ID" class="your-button-class">
  Book Appointment
</button>
<script src="https://yourdomain.com/widget.js"></script>
üí° Tip: Style the button however you want! The widget script only needs the data-booking-popup attribute to work.
Live Demo - Click the button

Type 3 Floating Button Widget

A persistent floating button that stays visible as users scroll your page. The easiest way to add booking functionality to any website.

<!-- Add this once at the bottom of your page -->
<script src="https://yourdomain.com/widget.js"
        data-provider-id="YOUR_USER_ID"
        data-float="true"
        data-button-text="Book Now"
        data-button-color="#0066FF"
        data-position="bottom-right">
</script>
üé® Customization Options:
  • data-button-text - Change button text (default: "Book")
  • data-button-color - Change button color (default: "#0066FF")
  • data-position - Position: "bottom-right", "bottom-left", "top-right", "top-left"
